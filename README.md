# TextEditor
Описание проекта:
---
Консольный текстовый редактор с возможностью
просмотра, редактирования и сохранения текста
в консоле.

## Модули
1. Универсальное меню
2. Консольный редактор
3. Консольный просмотрщик
4. Обозреватель файлов (опционально)
